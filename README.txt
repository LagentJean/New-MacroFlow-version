MacroFlow — Démo indépendante des masques musculaires

1. Ouvre index.html dans Chrome, Edge ou Safari.
2. Essaie les boutons Poitrine, Dos, Jambes et Full body.
3. Le curseur Temps écoulé fait évoluer rouge -> orange -> jaune -> vert.
4. Tu peux cliquer directement sur un muscle pour l'activer.
5. Le mode vérification rend les masques plus visibles pour juger le tracé.

Cette démo ne modifie pas l'application MacroFlow publiée.
